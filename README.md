Случайные переменные:
Случайная переменная — это переменная, которая принимает значения в зависимости от случайных событий. Она может 
принимать различные значения в зависимости от результата случайного эксперимента. Случайные переменные бывают двух типов:
    1. Дискретные случайные переменные:
        ◦ Принимают счетное количество значений.
        ◦ Например, количество выпавших орлов при подбрасывании монеты.
    2. Непрерывные случайные переменные:
        ◦ Принимают бесконечное количество значений в каком-то интервале.
        ◦ Например, время, необходимое для выполнения определенной задачи.
Распределение случайной переменной:
Распределение случайной переменной описывает вероятность того, что случайная переменная примет конкретное значение или
попадет в определенный интервал. Существует два основных типа распределений:
    1. Дискретные распределения:
        ◦ Равномерное распределение: Каждое значение имеет одинаковую вероятность.
        ◦ Бернуллиевское распределение: Используется для моделирования бинарных событий (например, успех/неудача).
        ◦ Распределение Пуассона: Описывает число событий, произошедших в фиксированном интервале времени или 
пространстве.
    2. Непрерывные распределения:
        ◦ Нормальное (гауссово) распределение: Широко используется из-за своих математических свойств и широкого 
применения в природе.
        ◦ Равномерное распределение: Все значения в интервале равновероятны.
        ◦ Экспоненциальное распределение: Часто используется для моделирования времени между независимыми событиями.
Выборка:
Выборка представляет собой подмножество элементов из генеральной совокупности. Генеральная совокупность — это полный 
набор объектов или событий, которые интересны для исследования. При проведении исследования часто сложно или невозможно
рассмотреть все элементы генеральной совокупности, поэтому используется выборка для делегирования представительности.
Процесс формирования выборки включает в себя различные методы, такие как случайное выборочное определение, 
стратифицированный выбор, кластеризованный выбор и другие. Цель выборки — сделать так, чтобы выборка 
была репрезентативной и отражала характеристики генеральной совокупности.

Вывод (или инференция) — это процесс делающий обобщения или выводы о генеральной совокупности на основе данных, 
полученных из выборки. То есть, на основе анализа данных выборки делаются утверждения или выводы относительно 
всей генеральной совокупности.
Основные методы вывода включают в себя точечную и интервальную оценку параметров, а также статистические тесты гипотез.
При этом важно учитывать различные искажения и ошибки, которые могут возникнуть при использовании выборки для вывода
о генеральной совокупности.

всех студентов на основе данных из выборки.
Важно проводить выборку и вывод таким образом, чтобы результаты были статистически значимыми и репрезентативными
для генеральной совокупности.
